# hlclock

[![Actions Status](https://github.com/nikosl/hlclock/workflows/CI/badge.svg)](https://github.com/nikosl/hlclock/actions)

Hybrid Logical Clocks

## Installation

### Using Opam

```bash
opam install inquire
```

### Using Esy

```bash
esy add @opam/inquire
```

## Usage

### In OCaml

```ocaml
let () = Hlclock.greet "World"
```
